Project 1: Simple Rule Based and Retreival Based Bot. 
RuleBasedAndRetreivalBot.ipynb containes 2 set of code where one is a simple rule based Chatbot where a bot from external planet want to know more about our planet. Secondly the retreival based bot uses the following steps of NLP:
1. Read text corpus
2. Preprocessing(stop words removal, Lowercase conversions etc)
3. Tokenization, Stemming and Lemmatization
4. Bag of words
5. One hot encoding
The Second bot uses the cosine similarity to fetch the appropriate responses from the corpus of text.
